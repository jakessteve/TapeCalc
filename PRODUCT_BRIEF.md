# Product Brief — TapeCalc v2

## One-Liner
Professional tape calculator desktop app with multi-tape management, unit/currency conversions, and function graphing — powered by Rust.

## Target Audience
Professionals, students, and engineers who need a fast, reliable desktop calculator with rich features beyond basic arithmetic.

## Key Differentiators
1. **Multi-tape system** — Manage multiple independent calculation tapes with full undo/redo history
2. **Rust-powered performance** — All computation happens in Rust for sub-50ms response times
3. **Comprehensive conversions** — 60+ units across 8 categories + live currency exchange rates
4. **Professional UX** — Three themes, full keyboard support, ARIA accessibility
