# SKILL: Content Strategy

**Trigger:** When @ba plans content for the product — blog posts, feature descriptions, help docs, cultural articles, or marketing campaigns.

---

## When to Use
- Planning a content calendar for sustained user engagement.
- Auditing existing content for gaps, staleness, or SEO issues.
- Aligning content with product launches (via `content-marketing` skill).
- Creating evergreen content for organic search traffic.
- Before `/go-to-market` workflow — content strategy feeds content creation.

---

## The 5-Step Content Strategy Process

### Step 1: Content Audit
What content exists? What's missing? What's stale?

```markdown
## Content Audit — [Date]
| Content | URL/Path | Status | Traffic | Last Updated | Action |
|---|---|---|---|---|---|
| [Title] | [URL] | Active/Stale/Missing | High/Med/Low | YYYY-MM-DD | Update/Keep/Remove/Create |
```

### Step 2: Strategy Alignment
Map content types to product goals:

| Goal | Content Type | Metric |
|---|---|---|
| **Acquisition** (new users) | SEO articles, tutorials, landing pages | Organic traffic, sign-ups |
| **Activation** (first value) | Onboarding guides, feature walkthroughs | Feature adoption rate |
| **Retention** (keep coming back) | Cultural articles, tips, newsletters | Return visits, DAU |
| **Revenue** (monetization) | Premium feature highlights, case studies | Conversion rate |
| **Referral** (word of mouth) | Shareable tools, infographics | Social shares, backlinks |

### Step 3: Editorial Calendar
Plan content cadence by type:

```markdown
## Editorial Calendar — [Month/Quarter]
| Week | Content Type | Topic | Target Keyword | Owner | Status |
|---|---|---|---|---|---|
| W1 | Feature guide | [Topic] | [Keyword] | @ba | Draft/Review/Published |
| W2 | Cultural article | [Topic] | [Keyword] | @ba | Draft |
| W3 | How-to tutorial | [Topic] | [Keyword] | @ba | Planned |
| W4 | Release notes | [Version] | — | @ba | Pending release |
```

### Step 4: Create Content
Use skill-specific workflows:
- **Blog posts / articles:** `content-marketing` skill
- **Landing page copy:** `seo-copywriting` skill
- **Technical docs:** `technical-writing` skill
- **Social threads:** `content-marketing` social template

## SEO Content Checklist
Before publishing any content:
- [ ] Target keyword identified and researched
- [ ] Title includes primary keyword (< 60 chars)
- [ ] Meta description compelling (< 155 chars)
- [ ] H1 includes keyword, proper heading hierarchy (H2 → H3)
- [ ] Content answers the user's search intent
- [ ] Internal links to related content (≥ 2)
- [ ] Images have alt text with relevant keywords
- [ ] Content length appropriate for topic (guides: 1000+, cultural: 800+)

## Rules
- Content must be **accurate** — especially astrological/cultural content. Verify with domain knowledge.
- **Write for the user first**, SEO second.
- **Update outdated content** rather than creating duplicates.
- **Every piece needs a CTA** — what should the reader do next?
- **Track everything** — if you can't measure it, you can't improve it.
